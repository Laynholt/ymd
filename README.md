# Установка
```
  git clone https://github.com/Laynholt/ymd.git
  cd ymd
  pip install virtualenv
  virtualenv venv
  .\venv\Scripts\activate
  pip install -r requirements.txt
  deactivate
```

# Запуск
```
  .\venv\Scripts\activate
  python ymd-r.py
  deactivate
```
